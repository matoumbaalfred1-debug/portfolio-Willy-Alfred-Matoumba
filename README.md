# Portfolio — MATOUMBA Willy Alfred

Portfolio personnel présentant mes projets, compétences et certifications.

## Stack technique

| Catégorie | Technologies |
|-----------|-------------|
| **Front-end** | HTML5, CSS3, JavaScript (Vanilla) |
| **Animations** | GSAP 3 (ScrollTrigger) |
| **Typographie** | Playfair Display, Plus Jakarta Sans (Google Fonts) |
| **Back-end** | Node.js, Express.js |
| **Bases de données** | PostgreSQL, MySQL |
| **DevOps** | Docker, Vercel, Netlify |
| **Langages** | Python, Java, JavaScript |
